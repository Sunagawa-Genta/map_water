## 紹介と使い方
湧き水APIを用いて、湧き水の場所が出てくる。

## 工夫した点
googlemapのAPIを使用したかったが何度か試したが、検索機能がうまく動かず断念。
その代わり、googlemapにピンを立てて、スクショしたものを載せている。
また、下部にリンク先があり、ピンを立てている場所のリストを共有できるようになっている。

## 苦戦した点
googlemapAPIの検索機能
なぜかうまくいかずつんでしまった。
湧き水APIの使い方、データをとってくるところもかなり苦戦したがなんとか自力でできた。
